# districtCenter

## What?
Given a set of id * district pairs, D, return the set D' of id * centroid pairs, st. centroid = (lat, long)

## What is included

* Blockgroups - All states
* Counties
    * Iowa
* Precincts
    * AK
    * LA
    * MA
    * MI
    * MN
    * NM
    * OH
    * PA
    * TX
    * UT
    * WI

* Blocks
